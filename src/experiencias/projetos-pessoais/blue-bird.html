História

Eu e meu amigo João Pedro decidimos refazer a interface do Twitter, hoje "X", só que com uma identidade visual nossa. Criamos uma página de login personalizada e a inicial(home). Foi pouco e nós paramos com o projeto por outras razões, mas valeu muito a pena porque pudemos vislumbrar como que funciona uma PullRequest, como é trabalhar com outra pessoa e isolar o ambiente de cada um em um branch separado para não correr o risco de prejudicar o branch de produção.


Desafios

Aprender os comandos de criação e alteração entre branches, git stash, pull e merge.
